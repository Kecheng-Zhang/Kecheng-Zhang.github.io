# 简介

博客是分享各种信息的网站，包括生活记录，或者是技术性的指南之类的。我开始想建立博客的原因其实很简单，就是感觉拥有自己的网站很酷。

我进行了一些搜索，最后决定使用Github pages + Jekyll来建立一个静态的博客网站。按照官网给的步骤加上网络上的教程，一个基本的网页很快就弄好了。
